About name 'ANOMS'

    Obviously, "ANOMS" is not an ordinary English word. Actually, "ANOMS" is originally an acronym for "ANOMS Is Not an Ordinary Minecraft Server".
    Well, I used to play minecraft on my desktop and build a server named "ANOMS". But later, I use "ANOMS" to call all the websites I've built, because I think this word is all-inclusive and can represent my exploration in computer science, especially in web server technology.

    Right now you can Interpret "ANOMS" like this:
      Active
      New
      Optimistic
      Motivated
      Sagacious
